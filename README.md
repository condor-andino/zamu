# ZAMU

**Reference mannequins for artists, right in your browser.**

ZAMU is a tool for posing human figures (male and female mannequins) and using them as references when drawing: you can adjust joints, rotate the camera, control perspective (lens/FOV), lighting, and shadows, and save your own poses. Everything works without installing anything and without an internet connection.

## 🎮 Try it out

👉 download the `index.html` file and open it by double-clicking in any browser. No server required.

## ✨ What can you do?

- Add multiple mannequins (male and female) and adjust their height.
- Edit each joint (head, neck, torso, shoulders, elbows, wrists, hips, knees, ankles).
- Predefined poses (standing, walking, running, jumping, squatting) and mirror any pose.
- Control the camera, perspective, and lens, as well as lighting, light direction, shadows, and the grid.
- Save your poses, **export** them to a file, and **import** them later.
- Interface available in 7 languages (Spanish, English, Korean, Chinese, Japanese, German, French) with light/dark themes.

## 🔒 Privacy

ZAMU **runs entirely on your device**. There’s no server, no accounts, and no data is sent or collected. Your poses are saved only in your browser’s cache, and you decide whether to export them for backup. Nothing leaves your computer.

## 🛠️ Technical Details

- A **single HTML file**, with no external dependencies or third-party libraries.
- Rendered using 2D Canvas in pure JavaScript (no frameworks).
- Works **offline** and is easy to audit: all the code is in plain sight.
